# @brunetco/contracts

Typed TypeScript client + types generated from the API's OpenAPI schema. **Do not hand-edit
generated files.**

- `openapi.json` — the API schema, exported from FastAPI by `scripts/export_openapi.py`.
  **Committed**, and the drift target CI checks (regenerate + diff must be clean).
- `src/generated/schema.ts` — types generated by `openapi-typescript`. Gitignored build artifact;
  regenerated by `pnpm --filter @brunetco/contracts generate`.
- `src/index.ts` — thin, hand-written typed fetch wrapper over the generated types.

Regenerate everything with `python make.py gen-contracts` (exports schema, then regenerates types).
